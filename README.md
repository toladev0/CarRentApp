# 🚗 CarRentApp

CarRentApp is a mobile application that allows users to browse and rent cars easily.  
The app helps users view available cars, check details, and make rental bookings.

## 📱 Features
- View available cars
- Car details page
- Rent or book a car
- Simple and user-friendly interface
- Navigation between screens

## 🛠️ Technologies Used
- Android Development
- Java
- Android Studio
- XML for UI design
